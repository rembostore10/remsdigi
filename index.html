Cara Order & Bantuan

Panduan singkat & jawaban pertanyaan umum

4 Langkah Order

1

Pilih Layanan

Tap salah satu menu di Beranda (Pulsa, Game, dll), pilih varian & isi data tujuan.

2

Bayar via QRIS

Scan QRIS dengan e-wallet apa saja (DANA, GoPay, OVO, ShopeePay, m-Banking). Wajib transfer pas dengan harga.

3

Upload Bukti

Screenshot bukti transfer & upload di form pembayaran. Tap Konfirmasi.

4

Tunggu Notifikasi

Cek menu Riwayat. Status akan otomatis berubah jadi Sukses / Gagal. Estimasi 1-15 menit di jam kerja.

Pertanyaan Umum

Berapa lama proses pesanan?
Pulsa, e-wallet, kuota: 1-5 menit. Token PLN & Game: 1-10 menit. Aplikasi Premium & Jasa CV: 10-60 menit. Di luar jam kerja (00:00-07:00) bisa lebih lama.
Bagaimana jika pesanan gagal / saldo tidak masuk?
Buka menu Live Chat (ikon balon kanan atas), kirim ID order Anda. Admin akan refund 100% atau memprosesnya ulang.
Bisa pesan dalam jumlah besar / reseller?
Bisa! Pilih menu "Join Reseller" di Beranda. Anda akan dapat harga reseller (lebih murah) untuk semua layanan.
Saya salah transfer (kelebihan/kurangan), gimana?
Hubungi admin via Live Chat dengan kirim foto bukti transfer. Selisih akan dikembalikan / diminta tambahan, tergantung kasus.
Apakah data saya aman?
Ya, data hanya disimpan di HP Anda (local storage browser). Server kami hanya menerima ID Order, target & bukti transfer untuk diproses oleh admin.
